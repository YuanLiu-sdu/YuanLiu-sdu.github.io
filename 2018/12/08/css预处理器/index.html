CSS预处理器

发表于 2018-12-08 | 更新于 2018-12-09

综述

预处理器特点

能力：

  • 嵌套，反映层级和约束
  • 变量和计算，减少重复代码
  • Extend和Mixin代码片段
  • 循环，适用于复杂的有规律的代码
  • import，CSS文件模块化，拆解文件易于维护

作用：帮助更好组织CSS代码，提高代码复用率，提升可维护性。
缺点：引入编译过程，有学习成本

变量

less与sass理念不同，less尽量使用css原有语法符号进行命名，如@符号，sass是为了避免混淆。使用css没有的$符号

Extend
